Introduction

Regression Model

Regression model describes how a variable x influences the value of another variable y

  • x: Independent / Explanatory variable
  • y: Dependent / Response variable

Linear Regression 是利用线性回归方程的最小二乘函数对一个或多个Independent & dependent变量之间关系进行建模的一种回归分析。 这种函数是一个或多个称为回归系数的模型参数的线性组合

回归分析： 了解两个或多个变量间是否相关，相关方向和强度。建立数学模型以便观察特定变量来预测研究者感兴趣的变量。通过回归分析可以由给出的自变量估计因变量的条件期望。

Relations between Variables

  • Deterministic: Function Realtion

    y=f(x)

    有精确和确定的关系

  • Nondeterministic: Statistical Relation

    y=f(x)+ϵ ϵ is an unkonwn perturbation(r.v)

    统计关系不是精确的关系，trend between the prtedictor x and the response y, there is also some scatter

Type of Relation

Linear

y=β0+β1x

  • If β1>0, positive linear relationship
  • If β1<0, negative linear relationship

The equation represents a stratight line in the plot.

 

Nonlinear

E.g: f(x)=log⁡(x),f(x)=x2+x+1

Absense of relation

f(x)=c f(x) does not depend on x

 

Measures of Linear Dependency

Covariance

Measure of linear dependency is the covariance.

cov(x,y)=∑i=1n(xi−x¯)(yi−y¯)n−1
  • Postive linear relation: cov(x,y) will be positive and large
  • Negative linear relation: cov(x,y) will be negative and large in absolute value
  • No relation/significantly linear: cov(x,y) will be close to zero

covariance depends on the units of measurement of variables.

Correlation Coefficient

Correlation Coefficient r does not depend on the units.

r(x,y)=cor(x,y)=cov(x,y)sxsysx2=∑i=1n(xi−x¯)n−1sy2=∑i=1n(yi−y¯)n−1
  • cor(x,y)∈[−1,1]
  • cor(x,y)=cor(y,x)
  • cor(ax+b,cy+d)=sign(a)sign(b)cor(x,y)

 

Simple Linear Regression

The simplest regression model is linear with a single predictor:

yi=β0+β1xi+ϵi

  • yi : the value of the dependent variable for the i-th observation

  • xi: the value of the independent variable for the i-th observation

  • ϵi: the error for the i-th observation

    E[ϵi]=0 σ[ϵ]=σ2

  • β0,β1: parameters / coefficients; intercept, slope

Goal: obtain estimates β0^,β1^ for β0,β1 to define regression line:

y^=β0^+β1^x

Provides the best fit for the data.

Residual: the difference between each value yi of the response variable and its estimation yi^

ei=yi−yi^

Hypothesis

Linearity

The existing relation between x and y is linear,

f(x)=β0+β1x

The data have to look reasonably straight

Homogeneity

The mean value of the error is zero.

E[ϵi]=0

Homoscedasticity

The variance of the error is constant.

Var[ϵi]=σ2

The dispersion of the data must be constant for the data to be homoscedastic.

b is heteroscedastic.

 

Independence

The observations are independent

E[ϵiϵj]=0

  • The data must be independent
  • An observation must not given information about the rest of the observations
  • Time series do not satisfy the independence hypothesis

 

Important Features

  1. The response yi in the i-th trial is a random variable:

    1. the constant term β0+β1x
    2. the random term ϵ
  1. E[y]=E[β0+β1x+ϵ]=β0+β1x+E[ϵ]=β0+β1x

  2. yi in the i-th trail exceeds or falls short of the value of the regression function by the error term amount of ϵ

  3. ϵi and ϵj are uncorrelated: σ[ϵi,ϵj]=0

  4. The error terms are assumed to be uncorrelated, so are the responses yi,yj

Estimation

Estimation of the regression coefficients and error variance

Method of Least Squares

To find good estimators of the regression parameters β0,β1: b0,b1/β0^,β1^

Seek to minimize:

Q=∑i=1n(yi−(β0^+β1^xi))2

Carful choice of β0^,β1^ are point estimator for β0,β1

 

 

  1. Numerical search procedures

  2. Analytical procedures. feasible when the regression model is not mathematically complex

    b1=∑(xi−x¯)(yi−y¯)∑(xi−x¯)2b0=1n(∑yi−b1∑xi)=y¯−b1x¯

Max(min)imization

Q=∑i=1n(yi−(β0^+β1^xi))2

Minimize this by maximing −Q

Find partials and set both equal to zero:

dQdb0=0dQdb1=0

Normal Equation

The result of maximization step are called the normal equations

∑yi=nb0+b1∑xi∑xiyi=b0∑xi+b1∑xi2

Solution of the two equations system:

b1=∑(xi−x¯)(yi−y¯)∑(xi−x¯)2b0=1n(∑yi−b1∑xi)=y¯−b1x¯

Estimation of the Variance

  • Variance of each observation yi is σ2 (same for error ϵ)
  • Each yi has different probability distribution with different means that depends on the level of xi

Maximum Likelihood estimator

The maximum likelihood estimator of σ2, it is a biased estimator

σ2^=∑i=1nei2n

Residual Variance

An unbiased estimator of σ2 is residual variance

SR2=∑i=1n(yi−y^)2n−2=SSEn−2=MSE

  • MSE is an unbiased estimator of σ2

    E[MSE]=σ2

  • The sum of squares SSE has n-2 degrees of freedom
